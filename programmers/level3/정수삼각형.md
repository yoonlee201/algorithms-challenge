### Submission
```java
class Solution {
    public int solution(int[][] triangle) {
        for (int h = triangle.length-2 ; h >= 0 ; h--){
            for (int i = 0; i < triangle[h].length; i++){
                triangle[h][i] += Math.max(triangle[h+1][i], triangle[h+1][i+1]);
            }
        }
        
        return triangle[0][0];
    } 
}
```
